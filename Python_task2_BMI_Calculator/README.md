# Task 2: BMI Calculator

A command-line Python application that calculates a user's Body Mass Index (BMI) and classifies it into standard health categories.

## Features
- Validates user input (rejects non-numeric and negative values).
- Calculates BMI based on weight (kg) and height (m).
- Classifies BMI into Underweight, Normal, Overweight, and Obese categories.
- Option to calculate another BMI without restarting the program.

## Tech Stack
- Python (Standard Libraries)
